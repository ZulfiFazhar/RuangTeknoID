## Instalasi

Ikuti langkah-langkah ini untuk menjalankan proyek secara lokal:

1. **Klon repositori:**
   ```bash
   git clone ttps://github.com/ZulfiFazhar/RuangTeknoID.git
   cd RuangTeknoID
   ```
2. **Install Dependencies:**

   ```bash
   npm install
   ```

3. **Jalankan Server:**
   ```bash
   npm run dev
   ```
